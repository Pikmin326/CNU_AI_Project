* 수정 사항
- 크롤링+임베딩과 질의 및 응답 파이프라인 분리
- 웹 인터페이스 구축
- json파일 말고 bin 파일로
- 벡터DB 추가
- DB서버 구축 및 연동
- 크롤링 페이지 넘어가기
- 저장된 문서 이후 문서만 크롤링
- 첨부파일 크롤링 및 파싱
- 온라인 웹 서버 구축

* 문제점
- 웹 크롤링 방식 변경
- 한정된 정보만 제공하는 문제(장학금에 대한 질의 시 여러 문서 중에서 가장 유사도가 높은 3문서만 나오는 문제)
- 답변이 너무 오래걸리는 문제 (평균 15초)
- 참고할 문헌이 하나일 경우 그 문헌이 세개로 나오는 문제
